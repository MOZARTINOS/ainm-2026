# AINM 2026

Private working repository for NM i AI / Norwegian AI Championship 2026.

## Goal
Ship fast, valid baselines first. Avoid zeros on any task.

## Structure
- `notes/` — live task notes, metrics, constraints, ideas
- `tracks/cv/` — computer vision task code
- `tracks/llm/` — language-model task code
- `tracks/ml/` — classical ML / tabular task code
- `shared/` — common utilities
- `submissions/` — generated submission artifacts
- `scripts/` — setup, eval, submit helpers

## Quick start
1. Open `notes/reveal.md` at kickoff.
2. Write down each task's input/output/metric/submission format.
3. Get one valid baseline for every task.
4. Only then optimize.
